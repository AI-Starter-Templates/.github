title: Image updates
description: "WUD (What's-Up-Docker) overlay: Discord ping when a base image has a newer tag."

import { Aside } from "@astrojs/starlight/components";

`WITH_WUD=1` brings up the What's-Up-Docker overlay: it watches every running container's image and pings a Discord webhook when a newer tag is available. Manual `docker compose pull` is still the apply step, so there are no surprise restarts.
