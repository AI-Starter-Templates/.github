---
title: Resource limits
description: Per-service CPU and memory limits, env-driven, sized for a 4-vCPU / 8 GB host by default.
---

import { Aside } from "@astrojs/starlight/components";

Every service in the compose stack has `deploy.resources.limits` and `reservations` driven by env vars. Defaults target a 4-vCPU / 8 GB single-host VPS; bump the relevant `*_CPUS` / `*_MEMORY` knobs in `compose/.env` for bigger boxes.
