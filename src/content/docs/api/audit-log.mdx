---
title: Audit log
description: Append-only audit trail for security- and compliance-relevant events.
---

import { Aside } from "@astrojs/starlight/components";

The api-template writes high-signal events (logins, permission changes, OAuth link/unlink, resource mutations on flagged tables) to an append-only `audit_log` table. The dashboard's "Recent activity" feed reads from it directly.
