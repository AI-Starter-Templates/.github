title: Observability
description: Prometheus + Grafana + Loki + Promtail overlay, with exporters for Postgres, Redis, and Traefik.
---

import { Aside } from "@astrojs/starlight/components";

`WITH_OBSERVABILITY=1` brings up the metrics + logs stack: Prometheus scrapes the API and service exporters, Grafana renders pre-provisioned dashboards, Loki + Promtail tail container logs. Zero per-event cost; lives on the same host.
