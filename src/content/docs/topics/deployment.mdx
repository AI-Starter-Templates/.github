The deployment story is intentionally boring: SSH into a VPS, install Docker, clone the three repos, fill in `compose/.env`, `STACK=prod ./scripts/compose-up.sh`. Traefik handles HTTPS via ACME; UFW + Cloudflare IP allowlist locks down the perimeter.
